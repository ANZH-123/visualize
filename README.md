# visualize
Get stream from sound card and draw waveform with matplotlib on python.

# Usage
```python
py audio_visual.py
```
or
```python
py audio_visualize_1d.py
```
or
```python
py audio_visualize_2d.py
```
or
```python
py audio_visualize_3d.py
```
