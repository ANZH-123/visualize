# visualize
Get stream from sound card and draw waveform with matplotlib on python.

# Usage
```bash
$ git clone https://github.com/darkchii/visualize.git
$ cd visualize
```
then
```bash
$ py run.py 1d
```
or
```bash
$ py run.py 2d
```
or
```bash
$ py run.py 3d
```
or
```bash
$ py run.py 1d /path/xxx.wav
```
or
```bash
$ py run.py 2d /path/xxx.wav
```
or
```bash
$ py run.py 3d /path/xxx.wav
```
